# szczytno-previews

Static-site previews of proposed websites for businesses in Szczytno (Warmia-Mazury, Polska). Generated by the leads workspace at `/Users/adrian/Claude/leads/`. Each business gets one folder at `<slug>/index.html` deployed via GitHub Pages.

These are working drafts, not finished products. The footer of each page makes that clear.

## Deployment

GitHub Actions builds on push to `main` and publishes the repo root to Pages. Live at <https://1nto5.github.io/szczytno-previews/>.

## Conventions

- Each `<slug>/index.html` is a single self-contained HTML file (inline `<style>`, no JS frameworks, no external fonts).
- `noindex,nofollow` on every page so previews don't end up in search results.
- File size budget per preview: under 100 KB.
- Hyphens only in all Polish text (no em/en dashes).
